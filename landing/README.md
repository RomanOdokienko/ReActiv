# ReActiv Landing

Изолированное приложение для разработки лендинга без правок существующих `frontend/` и `backend/`.

## Запуск

```bash
npm install --prefix landing
npm run dev --prefix landing
```

## Что важно

- Папка `landing/` живет отдельно от основного фронтенда.
- Здесь можно спокойно делать структуру, визуал и маркетинговые секции.
- Интеграцию в основной продукт лучше делать отдельным PR, когда лендинг уже стабилен.
